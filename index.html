Why Vaultkeep

Built to be simple, and safe by default

No accounts, no cloud, no monthly fee. Just a vault only you can open.

⚡

Auto-capture

Log into any site and Vaultkeep quietly picks up the credentials — no copy-pasting into a separate app.

🔐

AES-256 encryption

Every entry is encrypted with a key derived from your master password using PBKDF2 — 150,000 iterations.

📦

100% local storage

Data lives only in your browser's storage. There is no backend, no API, no server that could ever be breached.

🎲

Password generator

Create strong, random 16-character passwords in one click when signing up for new accounts.

👁

Show / hide / copy

Passwords stay masked by default. Reveal or copy to clipboard only when you choose to.

🧩

Open source

Every line of code is public on GitHub — read it yourself, fork it, or verify there's nothing sketchy going on.

Get started

Install in under two minutes

1

Download the code

Grab the ZIP from GitHub and extract it anywhere on your computer.

2

Open your browser's extensions page

Chrome or Edge: go to chrome://extensions (or edge://extensions).

3

Turn on Developer mode

You'll find the toggle in the top-right corner of the extensions page.

4

Click "Load unpacked"

Select the extracted vaultkeep folder. The extension icon will appear in your toolbar.

5

Set your master password

Click the icon, create a master password, and you're ready to go. Don't forget it — it can't be recovered.

Trust

We can't see your passwords. Neither can anyone else.

🛡️

Vaultkeep has no backend server, no analytics, and no network requests of any kind.

  • All data stored only in your own browser
  • Master password never leaves your device
  • No account signup, no email required
  • Fully open-source — verify it yourself
